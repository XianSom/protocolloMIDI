# MIDI Keyboard Project

## Overview
This project implements an interactive MIDI keyboard that allows users to click on keys to play notes and view the corresponding MIDI parameters for each note. The keyboard interface is designed to be user-friendly and visually appealing.

## Project Structure
```
midi-keyboard-project
├── src
│   ├── index.html         # Main HTML document for the MIDI keyboard interface
│   ├── styles
│   │   └── style.css      # Styles for the MIDI keyboard project
│   ├── scripts
│   │   └── keyboard.js     # JavaScript code for interactive functionality
├── assets
│   └── images
│       └── midi-keyboard.png # Image asset for the MIDI keyboard
├── package.json           # Configuration file for npm
└── README.md              # Documentation for the project
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd midi-keyboard-project
   ```

2. **Install Dependencies**
   If you have npm installed, run the following command to install any necessary dependencies:
   ```bash
   npm install
   ```

3. **Open the Project**
   Open `src/index.html` in your web browser to view the MIDI keyboard interface.

## Usage
- Click on the keys of the MIDI keyboard to play notes.
- The corresponding MIDI parameters for the selected note will be displayed on the interface.
